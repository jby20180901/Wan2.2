# Wan T2V A14B 架构详解与数据流分析

## 🏗️ 系统整体架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          WAN TEXT-TO-VIDEO A14B                              │
│                          (Wan T2V 14 Billion)                                │
└─────────────────────────────────────────────────────────────────────────────┘

                              ┌──────────────────┐
                              │  推理输入参数      │
                              ├──────────────────┤
                              │ • prompt: 文本    │
                              │ • size: 分辨率    │
                              │ • frame_num: 帧数 │
                              │ • sampling_steps  │
                              │ • guide_scale     │
                              └────────┬──────────┘
                                       │
                    ┌──────────────────┴──────────────────┐
                    │                                     │
           ┌────────▼────────┐              ┌────────────▼──────────┐
           │   T5 编码器      │              │  初始化随机噪声      │
           │  (UMT5-XXL)     │              │  (潜空间)            │
           └────────┬────────┘              └────────────┬──────────┘
                    │                                    │
    ┌───────────────▼──────────────┐      ┌──────────────▼──────────────┐
    │  文本特征                     │      │  初始潜码                   │
    │  (batch, text_len, 768)     │      │  (C=16, T=21, H=90, W=160)│
    │  ↓ 线性投影                  │      │  ↓ T5 的输出                │
    │  (batch, seq_len, 5120)     │      │                            │
    └───────────────┬──────────────┘      └──────────────┬──────────────┘
                    │                                    │
                    └──────────────┬─────────────────────┘
                                   │
                    ┌──────────────▼──────────────┐
                    │   40步去噪循环              │
                    │  (Diffusion Sampling)       │
                    └──────────────┬──────────────┘
                                   │
        ┌──────────────────────────┼──────────────────────────┐
        │                          │                          │
    ┌───▼──────┐            ┌────▼────────┐         ┌────────▼───┐
    │ 步骤 1-25 │            │  步骤 26-40   │         │  调度器     │
    │(高噪声)   │            │ (低噪声)     │         │  设置      │
    │high_noise │            │low_noise    │         │ timesteps  │
    │_model     │            │_model       │         └────┬───────┘
    │           │            │             │              │
    │ 引导系数:  │            │ 引导系数:    │         ┌────▼───────┐
    │ scale=4.0 │            │ scale=3.0   │         │时间步列表:  │
    │           │            │             │         │[999,880,...│
    │(结构生成) │            │(细节优化)   │         │ 0]        │
    └───┬──────┘            └────┬────────┘         └────┬───────┘
        │                        │                      │
        └────────────┬───────────┴──────────────────────┘
                     │
        ┌────────────▼────────────────────────────┐
        │   去噪预测循环                           │
        │   for step_idx, t in enumerate(times):  │
        │     ├─ latent_input = latents           │
        │     ├─ model_forward(latent, t, text)   │
        │     │  ├─ noise_pred_cond               │
        │     │  ├─ noise_pred_uncond             │
        │     │  └─ apply guidance                │
        │     ├─ scheduler.step()                 │
        │     │  (更新潜码)                      │
        │     └─ latents = updated_latents        │
        └────────────┬────────────────────────────┘
                     │
                     │ (循环40次)
                     │ step: 999→880→...→560→...→0
                     │ 逐步去噪
                     │
    ┌────────────────▼────────────────────┐
    │  最终潜码                           │
    │  (C=16, T=21, H=90, W=160)         │
    │  （完全去噪，对应最终视频）         │

        │  (C=3, T=81,      │
        │   H=720, W=1280)  │
        └────────┬──────────┘
                 │
         ┌───────▼────────┐
         │  输出视频文件  │
         │  或张量返回    │
         └────────────────┘
```

## 📊 核心组件架构

### 1. T5 文本编码器

```
                    ┌─────────────────────┐
                    │  文本输入            │
                    │  "一个女人唱歌"     │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │  T5 分词器          │
                    │  (SentencePiece)   │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────────────┐
                    │  Token IDs                  │
                    │  [101, 2054, 2572, ...]   │
                    │  (长度：1-512)             │
                    └──────────┬──────────────────┘
                               │
                    ┌──────────▼──────────────────────┐
                    │  T5 编码器 (UMT5-XXL)          │
                    │  - 24 层 Transformer           │
                    │  - 768 维隐藏层                │
                    │  - BF16 精度                   │
                    │  (25.7B 参数，仅编码器部分)    │
                    └──────────┬──────────────────────┘
                               │
                    ┌──────────▼──────────────────────┐
                    │  文本嵌入向量                   │
                    │  Shape: (batch=1, seq=512,     │
                    │         hidden=768)            │
                    │  ↓ 线性投影                    │
                    │  Shape: (batch=1, seq=512,     │
                    │         hidden=5120)           │
                    │  ↓ 交叉注意力中使用             │
                    └──────────────────────────────┘
```

### 2. Wan DiT (Diffusion Transformer)

```
                    ┌──────────────────────────────┐
                    │  输入: 潜码 + 文本条件         │
                    │  Shape: (1, 16, 21, 90, 160)│
                    │         (batch, C, T, H, W) │
                    └──────────┬───────────────────┘
                               │
                    ┌──────────▼───────────────────────┐
                    │  补丁嵌入 (Patch Embedding)      │
                    │  补丁大小: (1, 2, 2)             │
                    │  序列长度 = 21×45×80 = 75,600  │
                    │  补丁维度 = 16×1×2×2 = 64      │
                    │  → 线性映射到 5120 维            │
                    │  Output: (seq=75600, dim=5120) │
                    └──────────┬───────────────────────┘
                               │
                    ┌──────────▼──────────────────────┐
                    │  时间步嵌入 (Timestep Embed)    │
                    │  - 当前扩散步骤 t (0-1000)     │
                    │  - 正弦位置编码                │
                    │  - 维度: 256                  │
                    │  - 广播添加到序列               │
                    └──────────┬──────────────────────┘
                               │
                    ┌──────────▼──────────────────────────┐
                    │  40层 Transformer 块堆叠          │
                    │  每层包含:                       │
                    │  ├─ LayerNorm (RMSNorm)        │
                    │  ├─ 自注意力 (40头)             │
                    │  │  - Query, Key, Value        │
                    │  │  - 每头维度: 128             │
                    │  │  - QK 规范化                 │
                    │  ├─ LayerNorm                  │
                    │  ├─ 交叉注意力 (文本条件)      │
                    │  │  - Query 来自视频特征        │
                    │  │  - Key/Value 来自文本        │
                    │  │  - 交叉注意力规范化          │
                    │  ├─ LayerNorm                  │
                    │  └─ FFN (前馈网络)             │
                    │     5120 → 13824 → 5120        │
                    │     (扩展率: 2.7×)             │
                    │                                │
                    │  处理流程:                      │
                    │  x ← LayerNorm(x)              │
                    │  x ← x + SelfAttention(x)      │
                    │  x ← LayerNorm(x)              │
                    │  x ← x + CrossAttention(...)   │
                    │  x ← LayerNorm(x)              │
                    │  x ← x + FFN(x)                │
                    │                                │
                    │  (重复 40 次)                  │
                    └──────────┬───────────────────────┘
                               │
                    ┌──────────▼───────────────────────┐
                    │  输出头 (Output Projection)     │
                    │  - 线性层: 5120 → 64           │
                    │  (64 = 补丁中的元素数)         │
                    │  - 恢复为补丁格式              │
                    └──────────┬───────────────────────┘
                               │
                    ┌──────────▼───────────────────────┐
                    │  去补丁处理 (Unpatch)           │
                    │  补丁 (seq=75600, 64)           │
                    │  ↓ 重组                        │
                    │  噪声预测 (1, 16, 21, 90, 160)│
                    └──────────────────────────────────┘
```

### 3. VAE 编码/解码

```
编码过程:
┌─────────────────────────────────┐
│  输入视频 (RGB)                  │
│  (batch, 3, 81, 720, 1280)     │
│  或图像 (3, H, W)               │
└────────────┬────────────────────┘
             │
    ┌────────▼────────────────────┐
    │  标准化                      │
    │  使用预定义的 mean/std      │
    │  (独立为每个潜通道)         │
    └────────┬───────────────────┘
             │
    ┌────────▼───────────────────────┐
    │  VAE 编码器 (3D Encoder)      │
    │  - 3D 卷积层                 │
    │  - 时间/空间下采样            │
    │  - 步幅: (4, 8, 8)           │
    │  - 输出: mu, log_var         │
    └────────┬───────────────────────┘
             │
    ┌────────▼───────────────────────┐
    │  重参数化采样 (Reparameterize)│
    │  z = mu + sigma * epsilon     │
    │  epsilon ~ N(0, 1)           │
    └────────┬───────────────────────┘
             │
    ┌────────▼─────────────────────┐
    │  潜码 (Latent Code)          │
    │  (batch, 16, 21, 90, 160)   │
    │  压缩比: 13:1                │
    └───────────────────────────────┘

解码过程:
┌──────────────────────────────────┐
│  潜码 (Latent Code)              │
│  (batch, 16, T, H, W)           │
└────────────┬─────────────────────┘
             │
    ┌────────▼──────────────────────┐
    │  初始投影                     │
    │  线性层 16 → 64               │
    └────────┬──────────────────────┘
             │
    ┌────────▼──────────────────────────┐
    │  VAE 解码器 (3D Decoder)        │
    │  - 3D 转置卷积层              │
    │  - 时间/空间上采样             │
    │  - 步幅: (4, 8, 8)            │
    │  - 多层上采样                 │
    └────────┬──────────────────────────┘
             │
    ┌────────▼────────────────────────┐
    │  输出投影                      │
    │  通道映射到 3 (RGB)           │
    └────────┬────────────────────────┘
             │
    ┌────────▼────────────────────────┐
    │  范围裁剪 (Clamp)              │
    │  [-1, 1] 范围                 │
    └────────┬────────────────────────┘
             │
    ┌────────▼──────────────────────────┐
    │  反标准化                        │
    │  RGB 值 × std + mean             │
    └────────┬──────────────────────────┘
             │
    ┌────────▼──────────────────────────┐
    │  输出视频 (RGB)                   │
    │  (batch, 3, 81, 720, 1280)      │
    └──────────────────────────────────┘
```

## 🔄 推理数据流详细过程

### 第 0 步: 初始化

```
配置加载:
┌─────────────────────────────────────────┐
│  Config: wan_t2v_A14B                   │
├─────────────────────────────────────────┤
│  num_train_timesteps = 1000             │
│  sample_steps = 40                      │
│  sample_shift = 12.0                    │
│  boundary = 0.875 (模型切换点 = 875)   │
│  sample_guide_scale = (3.0, 4.0)        │
│  patch_size = (1, 2, 2)                 │
│  dim = 5120, ffn_dim = 13824            │
│  vae_stride = (4, 8, 8)                 │
└─────────────────────────────────────────┘

模型组件加载到GPU:
┌───────────────────┐  ┌────────────┐  ┌──────────────┐
│  T5 编码器        │  │ 高噪声模型 │  │ 低噪声模型   │

└─────────────────────────────────────────┘

采样调度器初始化:
┌─────────────────────────────────────────┐
│  FlowUniPCMultistepScheduler            │

```

### 第 1 步: 文本编码

```
输入:
┌──────────────────────────────────────────┐
│  提示词: "一个年轻女人在跳舞"           │
│  长度: 中文 16 个字符                   │
└──────────┬───────────────────────────────┘
           │
    ┌──────▼─────────────────────────┐
    │  T5 分词                       │
    │  → ~16 个中文 token IDs       │
    │  → 补充到 512 个 token        │
    │  Shape: (1, 512)             │
    └──────┬───────────────────────┘
           │
    ┌──────▼──────────────────────────┐
    │  T5 编码器前向                  │
    │  24 层 Transformer (UMT5-XXL)  │
    │  输出形状: (1, 512, 768)       │
    │  → 只取[有效部分]              │
    └──────┬───────────────────────┘
           │
    ┌──────▼────────────────────────┐
    │  线性映射                      │
    │  768 → 5120                   │
    │  输出形状: (1, 512, 5120)     │
    │  ↓ 用作 cross-attention       │
    │  (所有 40 层)                 │
    └───────────────────────────────┘
```

### 第 2 步: 初始化潜码

```
参数计算:
  frame_num = 81
  size = (1280, 720)
  vae_stride = (4, 8, 8)
  patch_size = (1, 2, 2)

  T = (81 - 1) / 4 + 1 = 21
  H = 720 / 8 = 90
  W = 1280 / 8 = 160
  C = 16 (VAE 潜通道)

初始化:
┌──────────────────────────────────────┐
│  随机噪声采样                        │
│  使用种子确保可重复                 │
│  Shape: (16, 21, 90, 160)           │
│  分布: N(0, 1) - 高斯分布            │
│  设备: GPU                           │
│  精度: float32 → bfloat16 (自动)    │
└──────┬───────────────────────────────┘
       │
┌──────▼────────────────────────┐
│  初始潜码 (纯噪声)            │
│  latents[0] = noise           │
│  状态: 完全随机 (99.9% 噪声)  │
└───────────────────────────────┘
```

### 第 3 步: 采样循环 (40 次迭代)

```
for step_idx in range(40):
    t = timesteps[step_idx]  # 999 → 880 → ... → 0
    
    ┌─────────────────────────────────────────┐
    │  步骤 {step_idx}                        │
    │  时间步: t = {t}                        │
    │  噪声水平: {t/1000}%                   │
    └────────┬────────────────────────────────┘
             │
    ┌────────▼────────────────────────────┐
    │  第 1 阶段: 获取文本条件化预测      │
    │                                    │
    │  输入:                              │
    │  ├─ 潜码 (16, 21, 90, 160)        │
    │  ├─ 时间步 t                      │
    │  └─ 文本条件 (512, 5120)          │
    │                                    │
    │  模型选择:                          │
    │  if t >= 875:                      │
    │      使用 high_noise_model         │
    │  else:                             │
    │      使用 low_noise_model          │
    │                                    │
    │  前向传播:                          │
    │  noise_pred_cond =                 │
    │    model(latent,                   │
    │          t,                        │
    │          context=text_emb)         │
    │                                    │
    │  输出形状: (16, 21, 90, 160)      │
    └────────┬────────────────────────────┘
             │
    ┌────────▼────────────────────────────┐
    │  第 2 阶段: 获取无条件预测          │
    │  (使用负提示词代替文本)            │
    │                                    │
    │  负提示词嵌入:                      │
    │  "色调艳丽, 过曝, ..."             │
    │  ↓ T5 编码                         │
    │  文本特征 (512, 5120)              │
    │                                    │
    │  前向传播:                          │
    │  noise_pred_uncond =               │
    │    model(latent,                   │
    │          t,                        │
    │          context=neg_emb)          │
    │                                    │
    │  输出形状: (16, 21, 90, 160)      │
    └────────┬────────────────────────────┘
             │
    ┌────────▼─────────────────────────────┐
    │  第 3 阶段: 应用分类器无关引导      │
    │  (Classifier-Free Guidance)        │
    │                                    │
    │  选择引导尺度:                      │
    │  if t >= 875:                      │
    │      guide_scale = 4.0             │
    │      (强调结构)                    │
    │  else:                             │
    │      guide_scale = 3.0             │
    │      (细节优化)                    │
    │                                    │
    │  计算引导后的预测:                  │
    │  noise_pred = noise_pred_uncond +  │
    │    guide_scale *                   │
    │    (noise_pred_cond -              │
    │     noise_pred_uncond)             │
    │                                    │
    │  效果:                              │
    │  noise_pred_uncond:                │
    │    ↑ 自由创意                      │
    │  noise_pred_cond - uncond:         │
    │    ↑ 文本约束方向                  │
    │  乘以 guide_scale:                 │
    │    ↑ 控制约束强度                  │
    └────────┬─────────────────────────────┘
             │
    ┌────────▼──────────────────────────┐
    │  第 4 阶段: 调度器更新步骤        │
    │  (使用采样求解器)                 │
    │                                  │
    │  调用 scheduler.step():           │
    │  输入:                            │
    │  ├─ noise_pred (16, 21, 90, 160)│
    │  ├─ timestep t                  │
    │  └─ sample (当前潜码)           │
    │                                  │
    │  算法: UniPC (高效求解器)        │
    │  输出新的潜码:                    │
    │  latents_new =                   │
    │    scheduler.step(               │
    │      noise_pred,                 │
    │      t,                          │
    │      latents)[0]                 │
    │                                  │
    │  效果:                            │
    │  - 去除预测的噪声                │
    │  - 逐步向最终图像靠近            │
    │  - 潜码质量逐步提高              │
    └────────┬──────────────────────────┘
             │
    ┌────────▼──────────────────────────┐
    │  第 5 阶段: 更新潜码              │
    │                                  │
    │  latents = latents_new           │
    │                                  │
    │  状态:                            │
    │  去噪进度 = (40 - step_idx) / 40  │
    │  从 97.5% 逐步减少到 0%           │
    └──────────────────────────────────┘

(循环 40 次结束)
```

### 第 4 步: 最终 VAE 解码

```
┌───────────────────────────────────┐
│  最终潜码                         │
│  (完全去噪)                      │
│  Shape: (16, 21, 90, 160)       │
│  信息: 81 帧视频的压缩表示       │
└───┬─────────────────────────────┘
    │
┌───▼──────────────────────────────┐
│  VAE 解码器                       │
│  前向传播: decode([latents])     │
│                                  │
│  处理流程:                        │
│  1. 通道映射: 16 → 64            │
│  2. 初始卷积块                  │
│  3. 4 层上采样: 每层翻倍分辨率   │
│  4. 时间上采样: 4× 帧率        │
```

## 🎯 模型选择逻辑

```
时间步 t
    │
    ├─ 如果 t >= 875 (= 0.875 × 1000)
    │   ↓
    │   使用 high_noise_model
    │   特征: 生成大尺度结构
    │   引导系数: 4.0 (强)
    │   步骤: ~25 步 (t: 999→875)
    │   占比: 87.5%
    │
    └─ 如果 t < 875
        ↓
        使用 low_noise_model
        特征: 优化细节纹理
        引导系数: 3.0 (中)
        步骤: ~15 步 (t: 875→0)
        占比: 12.5%

双模型策略优势:
✓ 结构阶段用专门模型 → 更好的大框架
✓ 细节阶段用专门模型 → 更好的清晰度
✓ 避免单模型过度拟合某个噪声水平
✓ 充分利用不同模型的优势
```

## 💾 内存占用估算

```
单个组件峰值内存:

T5 编码器:
├─ 模型参数: 25.7B × 2 bytes (BF16) ≈ 51 GB
└─ 活动内存: ~20 GB (在 CPU 或单 GPU)

高噪声模型 (40 层 DiT):
├─ 参数: ~14B × 2 bytes ≈ 28 GB
├─ 激活值: seq=75600, dim=5120, 40层
│  ~ 75600 × 5120 × 4 × 2 ≈ 3 GB
└─ 总计: ~30 GB

低噪声模型:
└─ 同上: ~30 GB

VAE 解码器:
└─ 参数 + 激活: ~5 GB

总 VRAM: ~60-80 GB (对于 A100/H100)

优化策略:
- 启用 offload_model: T5 和模型交替在 CPU/GPU
- 使用 FSDP: 分布式并行
- 使用 Sequence Parallel: 序列长度并行
```
